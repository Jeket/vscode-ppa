# Visual Studio Code PPA

## Contains
* Visual Studio Code
* Visual Studio Code Insiders

## Installation
add **vscode.list** to **/etc/apt/sources.list.d** 

sudo wget -O /etc/apt/sources.list.d/vscode.list https://github.com/tagplus5/vscode-ppa/raw/master/vscode.list
